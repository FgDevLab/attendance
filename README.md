## Development Setup

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd <repository-name>
   ```

2. **Create ENV file**
   
   in frontend directory create `.env` file with following format:
   
   ```bash
   VITE_BACKEND_API_URL=http://backend-api-url
   VITE_PORT=default 7100
   ```
